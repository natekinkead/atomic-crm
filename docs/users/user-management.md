# User Management

## Overview

This document explains how to manage users in Atomic CRM.

## Creating the First User

When first opening the Atomic CRM App, users are prompted to create the initial account. "This first user will automatically be granted administrator role."

**Note:** If the application is connected to the production database, the first user must validate their account by mail.

## Adding Users

"Any user with the administrator role can create new users."

**Process:**
- Click the avatar in the top right corner
- Select 'Users'
- From this page, you can add new users, edit existing users, and delete users

A visual guide is included showing the user interface for adding users.

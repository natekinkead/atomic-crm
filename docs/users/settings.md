# Settings

## Application Settings

Admin users can customize the interface, define categories, and more.

## Accessing the Settings Page

Log in as an admin, then click your profile picture in the top right corner and select "Settings" from the dropdown. The settings page uses a left sidebar for navigation between sections.

## Changing Application Settings

The Settings page allows customization of:

- Application name and logo
- Company sectors
- Deal pipeline stages (prospecting, negotiation, closed-won, etc.)
- Deal categories
- Note statuses (hot, cold, etc.)
- Task types (call, email, etc.)

Changes save immediately and appear across the application. For instance, updating deal stages makes them available in the Kanban view and when creating or editing deals.

Renaming an existing category updates all associated records. For example, renaming "Prospecting" to "Lead" moves all deals from that stage to the new name.
